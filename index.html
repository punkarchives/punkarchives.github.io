    (05/26/25) Biggest Update By Far

    This update has been a work in progress since the start of the site. You can finally edit and add information yourself! No more filling out and submitting a form then waiting for it to get added. All you have to do is log in, go to a page, and press edit. We also now have a logo, and the "points" leaderboard was replaced by a "band's added" leaderboard. Some of the new features might be a little confusing at first so I'll explain a few. When a band is flagged, it means it's outdated or inaccurate. You can see these flags in the bands list, and you can toggle them on the band page. To edit a page, look at the title (eg: Black Flag) and there will be an edit button to the right of it. Click that and you'll be brought to the edit page. After you edit some details, they should save and publish instantly. In the case of someone messing up a release's info, you can flag the release as 'Delete' or 'Restore'. If the release isn't real and someone only created it to screw around, then you should mark it as 'Delete'. If someone edited the info of an already existing release, you should mark it as 'Restore'. These release flags exist for admins to come and fix things, but users are welcome to help with restoration too.

    What Is Punk Archives?

    Punk Archives is basically just the punk equivalent of metal-archives. Anyone can (and is encouraged to) submit info on any punk-related bands.

    Important Features To Know:

    •Refresh with ctrl+shift+r if changes aren't showing.
    •Hover your mouse over a release's name to see the image.
    •In All Bands, you can search by artist name, record label, genre, location, and release name.
    •Bands marked with a red flag are either incomplete or inaccurate.
