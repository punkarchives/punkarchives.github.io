Editor Help

How do I edit something?

Make sure you're logged into your account. Then, go to the band, label, or zine page and look for the Edit button next to the title. Click it to open the edit page.

How do I add a new band, label, or zine?

Go to the “Add New” page. Before adding a band, make sure you put "the" at the end of the band name (eg: Clash, The). This rule only applies to band names. Labels and zines can start with "The". Now choose what you're adding, enter the name, and click the button to submit. It will be automatically added and you can edit it from its page.

I made a typo in the band name, what do I do?

First, make sure not to edit anything. If you already edited something and don’t want to add it all again, a moderator can transfer the data for you. Add the band again, but make sure it’s spelled correctly. Then go to the band page with the typo and press “contact moderator”. The misspelled band will be deleted and its info will be transferred over if necessary.

How do I add an image to something?

First, you need the image URL. If you're not sure how to get that:
• Find the image online
• Right-click it
• Choose "Copy image address"
Then paste that link where it's required.

Why isn't the image I added displaying?

If it’s an album cover, it won’t automatically show. To see it, hover over the album title or click "Download Image". If it still doesn’t appear, it's probably an issue with the link (most common) or your internet.

Why isn't a member I added showing? / How do I properly set the status of a member?

Band members won’t show unless their status is set to one of these:
• Current: member is still in the band and the band is active
• Former: member is no longer in the band and wasn’t part of the last lineup
• Last Known Lineup: member was in the last known lineup before the band went inactive

How do I add multiple versions of a release?

Go to the page of your specified release (not the band page), press edit, and press “Add Extra Version”

What do I do if an album was released by multiple record labels?

In a scenario where 2 different labels put out the same pressing of a release, you can put multiple labels separated by commas on the release info. If different pressings of a release were put out by different labels, then you can just specify that when adding another version of a release.

How do I add the listen button to a release?

Get a YouTube link for the release, and paste it into the “YouTube Link” field in the band’s edit page.

Why aren't a band's releases sorting by date?

Currently the sorting feature only supports years. If the release date is anything other than a 4-digit year, then it won't get sorted.

How does "Name" and "Real Name" work for band members?

The name field is reserved for the name the member is known by while performing in that band. This could be a stage name, like Johnny Rotten, or just the member’s real name. The real name field has to be the member’s real name, with no exceptions. This is so the member’s individual page that lists all of the bands they were in works without counting stage names as separate people.

Is this band punk enough to be added?

I don’t really care about gatekeeping here. There will never be a large debate about whether a band can be added or not. As long as you’re adding a band with good intentions, and they aren’t collectively agreed upon as not punk (like pop or country), then go ahead. I’ve even let Rare Americans slide, but that was about my limit.

Are Nazi bands allowed?

You can’t add upcoming shows for these bands or support their ideologies, but archiving them is allowed. Don't abuse this by spamming the site with these bands though. This primarily exists so bands who only became controversial later-on can still be added. Also note that obviously satire bands like Anal Cunt are not Nazi bands.

Is NSFW allowed?

NSFW is allowed when archiving (as long as it won't get anyone in legal trouble), so don’t worry about your goregrind album covers or noisecore song titles. This of course is restricted to archiving though. You can’t just go and make one of the images on the home page hardcore porn.

What is flagging a release?

On the editor page, you can flag releases as Delete or Restore.
• Delete: A moderator will review and remove the release.
• Restore: A moderator will revert the release to an earlier version.
After flagging a release select the “Request Moderator” button if available.
Anyone with the right info can also restore it manually. To remove a flag, just change the value to anything other than "Delete" or "Restore."

What is flagging a band?

If a band is flagged it means the info on the page is incomplete, outdated, or just plain wrong. On the "All Bands" page, you'll see a flag icon next to the band name if it's been flagged. You can toggle this flag from the band's page. Also note that you shouldn’t flag bands as incomplete if the missing info can’t be found anywhere.

How do I delete something?

Users can't delete anything directly. To delete a release, see "What is flagging a release?". To delete a band, set the description to “delete this band” and select the “Request Moderator” button. To delete anything else, just rename it to something like "delete" or "please delete this" and select the “Request Moderator” button if available.

How do I earn points?

Points are earned by adding bands and releases. These actions give you 1 point each. The main purpose of these points is just to show how active you are on the site, though the top 3 users get to control an image at the bottom of the home page.
