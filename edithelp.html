Editor Help

How do I edit something?

Make sure you're logged into your account. Then, go to the band, label, or zine page and look for the Edit button next to the title. Click it to open the edit page.

What is flagging a release?

On the editor page, you can flag releases as Delete or Restore.
• Delete: A moderator will review and remove the release.
• Restore: A moderator will revert the release to an earlier version.
Anyone with the right info can also restore it manually. To remove a flag, just change the value to anything other than "Delete" or "Restore."

What is flagging a band?

Flagging a band means the info on the page is incomplete, outdated, or just plain wrong. On the "All Bands" page, you'll see a flag icon next to the band name if it's been flagged. You can toggle this flag from the band's page.

How do I delete something?

Users can't delete anything directly. To delete a release, see "What is flagging a release?". To delete anything else, just rename it to something like "delete" or "please delete this" and a moderator will deal with it.

How do I add a new band, label, or zine?

Go to the Add New page. Choose what you're adding, enter the name, and click the button to submit. It will be automatically added and you can edit it from its page.

How do I add an image to something?

First, you need the image URL. If you're not sure how to get that:
• Find the image online
• Right-click it
• Choose "Copy image address"
Then paste that link where it's required.

Why isn't the image I added displaying?

If it’s an album cover, it won’t automatically show. To see it, hover over the album title or click "Download Image". If it still doesn’t appear, it's probably an issue with the link (most common) or your internet.

Why isn't a member I added showing? / How do I properly set the status of a member?

Band members won’t show unless their status is set to one of these:
• Current: member is still in the band and the band is active
• Former: member is no longer in the band and wasn’t part of the last lineup
• Last Known Lineup: member was in the last known lineup before the band went inactive

Why aren't a band's releases sorting by date?

Currently the sorting feature only supports years. If the release date is anything other than a 4-digit year, then it won't get sorted.

How do I add the listen button to a release?

This feature is currently unable to be edited by users. A moderator will occasionally go and add some manually. If you really want a release to have it, then you can use the contact button to ask.
